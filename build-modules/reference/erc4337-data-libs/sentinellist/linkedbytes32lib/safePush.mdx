---
title: "safePush"
---

Add an item to the list and initialize it if it is not initialized.

## Usage

```solidity
list.safePush(bytes32(keccak256("newEntry")));
```

## Parameters

### self

- Type: `LinkedBytes32`

The storage pointer of the list.

### newEntry

- Type: `bytes32`

The entry to be added to the list.

## Returns

None
