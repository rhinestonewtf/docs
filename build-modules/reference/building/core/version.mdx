title: "version"

Get the version of the module.

## Usage

```solidity
/**
* The version of the module
*
* @return version The version of the module
*/
function version() external pure returns (string memory) {
    return "0.0.1";
}
```

## Parameters

None

## Returns

### version

- Type: `string`

The version of the module.
