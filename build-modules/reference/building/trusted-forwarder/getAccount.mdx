---
title: "_getAccount"
---

Get the address of account when using the [`TrustedForwarder`](https://github.com/rhinestonewtf/module-bases/blob/main/src/utils/TrustedForwarder.sol) to allow modules to be called by other modules rather than directly by the account. Inherited from `TrustedForwarder`.

## Usage

```solidity
address account = _getAccount();
```

## Parameters

None

## Returns

### account

- Type: `address`

The address of account.
