---
title: "installModule"
---

Installs a module into the account.

## Usage

```solidity
instance.installModule({
    moduleTypeId: MODULE_TYPE_VALIDATOR,
    module: validator,
    data: validatorData
});
```

## Parameters

### instance

- Type: [`AccountInstance`](../../glossary/types#accountinstance)

The instance of the account used for testing.

### moduleTypeId

- Type: `uint256`

The type of the module to install.

### module

- Type: `address`

The address of the module to install.

### data

- Type: `bytes`

The data to send with the module installation.

## Returns

### userOpData

- Type: [`UserOpData`](../../glossary/types#useropdata)

The data of the UserOperation.
