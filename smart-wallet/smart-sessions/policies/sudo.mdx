---
title: "Sudo Policy"
sidebarTitle: "Sudo"
---

Sudo policy allows any transaction to pass. It is the default session.

To enable a sudo policy:

```ts {6-14}
const session: Session = {
  owners: {
    type: 'ecdsa',
    accounts: [sessionOwnerAccount],
  },
  actions: [
    {
      policies: [
        {
          type: 'sudo',
        },
      ],
    },
  ],
}
```
