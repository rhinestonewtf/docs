sidebarTitle: "Usage limit"
---

Limit how many times a session function can be called.

```ts {13}
const session = toSession({
  chain: base,
  owners: {
    type: 'ecdsa',
    accounts: [sessionOwnerAccount],
  },
  permissions: [
    {
      abi: erc20Abi,
      address: usdcAddress,
      functions: {
        transfer: {
          maxUses: 10n,
        },
      },
    },
  ],
})
```

This caps the function at 10 calls.
