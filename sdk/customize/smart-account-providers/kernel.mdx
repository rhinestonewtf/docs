---
title: "Kernel"
---

To create a [ZeroDev Kernel](https://github.com/zerodevapp/kernel) smart account:

```ts {2-4}
const rhinestoneAccount = await createRhinestoneAccount({
  account: {
    type: 'kernel',
  },
  owners: {
    type: 'ecdsa',
    accounts: [account],
  },
  rhinestoneApiKey,
})
```
